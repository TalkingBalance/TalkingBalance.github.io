Contact

If you'd like to contact us, please click here to send us an email. If you're reporting an issue with the program, please try to be as specific as possible. For example if the scales can't connect please tell us which scales you have, which adaptors you're using and what fixes you've tried to avoid us having to ask for this information before we can assist you.
